# DiscoverLanka – Frontend Travel Website

DiscoverLanka is a Sri Lanka travel discovery and trip-planning frontend website. It is designed to help users explore destinations across the island, browse travel categories, read traveler reviews, and build a printable itinerary through a clean, modern interface. :contentReference[oaicite:0]{index=0}

## Project Overview

The website is built around the idea of making Sri Lanka travel planning easier and more inspiring. On the homepage, users can explore destination categories such as Beaches, Hill Country, Waterfalls, Heritage Sites, and Wildlife, then move into planning tools like the smart trip planner and manual itinerary planner. :contentReference[oaicite:1]{index=1}

This frontend project also includes supporting pages such as About, Reviews, Contact, Blog, Help Center, Privacy Policy, and Terms of Service, giving the site the structure of a complete travel platform. :contentReference[oaicite:2]{index=2}

## Main Features

### 1. Homepage
The homepage introduces DiscoverLanka as a Sri Lanka travel website where users can explore destinations and turn them into a printable itinerary. It includes sections such as:
- Hero section with call-to-action buttons
- Explore Sri Lanka by category
- How It Works
- Popular Destinations
- Traveler testimonials
- Footer with support and navigation links :contentReference[oaicite:3]{index=3}

### 2. Destination Categories
Users can browse major travel categories directly from the navigation and homepage:
- Beaches
- Hill Country
- Waterfalls
- Heritage Sites
- Wildlife :contentReference[oaicite:4]{index=4}

### 3. Manual Itinerary Planner
The site includes a dedicated manual itinerary planner where users can build their trip by entering:
- Trip title
- Duration
- Starting and ending locations
- Budget
- Travel mode
- Accommodation
- Interests
- Food preference
- Special notes
- Main destinations
- Stops on the way
- Day-by-day plan

It also includes itinerary preview and print functionality. :contentReference[oaicite:5]{index=5}

### 4. Reviews Page
The reviews section allows visitors to:
- Read featured traveler stories
- See review statistics
- Submit their own travel review
- Filter reviews by rating :contentReference[oaicite:6]{index=6}

### 5. About and Help Pages
The About page explains that DiscoverLanka is a university student project in development, focused on destination discovery and trip-planning ideas rather than a finished commercial travel service. The Help Center answers common questions about the website and its current development stage. :contentReference[oaicite:7]{index=7}

## How to Run the Project

1. Download or clone the project files.
2. Open the project folder in VS Code or any code editor.
3. Open `index.html` in your browser.

For a better development experience, run it with a local live server extension. This step is general guidance for static frontend projects. :contentReference[oaicite:11]{index=11}
